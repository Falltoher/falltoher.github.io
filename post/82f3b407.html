简介

requests是一个Python第三方库，处理URL资源特别方便。它的目标是简化HTTP请求，使开发人员能够快速进行HTTP请求。

请求头

请求头通常会加 UA（user agent），这个主要是模仿浏览器的行为，以防止被网站屏蔽。

参数

  1. get请求参数
    get 请求的参数可以直接在url后面加参数，url?key1=value1&key2=value2，即 url 后面加问号，然后紧接着多个参数的键和值，多个键值之间用 & 符号链接；
    这种方式简单是简单，但是参数多了之后，url 会变得很长，看起来胀眼睛，为了更好的可读性，requests 支持这样传递 get 请求的参数：
  2. post请求参数
    post 请求参数一般是通过data参数传递，通常 data 是一个字典形式 如果你拿到的参数，是一个 json 格式，可以直接传递给 json 参数
