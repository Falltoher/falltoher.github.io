Redis

Created2023-04-11|Updated2023-05-04|后端
|Word count:405|Reading time:1min|Post View:

概述

Redis 是一款高性能的 NoSQL 系列的非关系型数据库
优点：

  1. 速度快
  2. 支持丰富的数据类型
  3. 持久化
  4. 支持集群
  5. 支持事务
  6. 支持 Lua 脚本
  7. 支持多种语言

缺点：

  1. 单线程
  2. 内存消耗大
  3. 持久化效率低
  4. 集群规模有限

应用场景：

  • 缓存（数据查询、短连接、新闻内容、商品内容等）
  • 计数器
  • 消息队列
  • 分布式锁
  • 排行榜
  • 社交网络
  • 实时系统

数据结构

Redis 的数据结构：
Redis 存储的是 key-value 格式的数据，其中 key 都是字符串，value 有 5 种不同的数据结构：

  • 字符串类型 String
  • 哈希类型 Hash：map 格式
  • 列表类型 List：LinkedList 格式（支持重复元素）
  • 集合类型 Set：HashSet 格式（不允许重复元素）
  • 有序集合类型 SortedSet：不允许重复元素，且元素有顺序
